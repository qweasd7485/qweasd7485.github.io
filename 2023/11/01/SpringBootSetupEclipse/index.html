使用Eclipse開發Spring Boot網站專案

發表於 2023-11-01 更新於 2023-11-23 分類於 Spring Boot相關

運行開發環境與網頁技術

OS:Windows 10 Professional 64bit

IDE:Eclipse IDE for Eclipse Committers

Web:Spring Boot

Code:Java

Project:Maven Project

步驟

1.Eclipse若要開發Spring Boot專案，需到套件市集將開發Spring Boot的套件下載並安裝；開啟Eclipse後->上方功能區Help->選項Marketplace...：

2.搜尋關鍵字Spring後，出現Spring Tools 4(如紅框處)，進行安裝：

3.套件安裝完畢後，開始建立專案；Spring Boot建立專案的方式有兩種：

3-1.Spring 的官方網站提供了實用的工具，叫做「Spring Initializr」。它能夠幫助我們快速地產生全新的 Spring Boot 專案，先到該頁面，以下是它的頁面：

網站聯結：Spring Initializr

Project 專案管理方式 Maven
Language 程式語言 Java

Packaging 專案壓縮產出副檔格式 Jar
Java 版本 11
