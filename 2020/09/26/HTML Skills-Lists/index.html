※、Type of bullet項目符號

可以於<ul>定義type屬性數值以設定項目符號的樣式，一般預設為disc(黑色實心圓點)；基本可設定參數有三個：

參數設定值 說明
squary 黑色實心正方形
disc 黑色實心圓點
circle 黑色空心圓圈

若要使用其他符號可以使用css中的list-style-type。

※、Type of bullet(Ordered Lists)順序列表的項目符號

同理，可以於<ol>定義type屬性數值以設定順序項目符號的樣式，一般預設為1(阿拉伯數字)；可設定參數如下：

參數設定值 說明
<ol type = “1”> 阿拉伯數字
<ol type = “I”> 大寫羅馬數字
<ol type = “i”> 小寫羅馬數字
<ol type = “A”> 大寫英文字母
<ol type = “a”> 小寫英文字母
